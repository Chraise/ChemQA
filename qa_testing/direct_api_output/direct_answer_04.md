### 2. 技术细节  
#### (1) 反应机理与催化过程  
羰基还原遵循质子耦合电子转移（PCET）路径：  
```  
R₁R₂C=O + e⁻ → [R₁R₂C-O]⁻*  (吸附)  
[R₁R₂C-O]⁻* + H⁺ → R₁R₂CH-O*  
R₁R₂CH-O* + e⁻ + H⁺ → R₁R₂CH₂ + H₂O  
```  
**选择性决速步**：首电子转移后形成的碳氧自由基阴离子（*R-C=O⁻）稳定性受催化剂d带中心位置调控（d带中心下移0.2 eV可使吸附能降低1.5 eV，见*Science 2017, 355, 146*）。
